## Utils

This repository implements a series of self-used C++ standard library components.

*   **Containers:**
    *   array
    *   vector
    *   list
    *   map
    *   set

*   **Smart Pointers:**
    *   shared_ptr
    *   unique_ptr

*   **General Utilities:**
    *   function
    *   optional
    *   variant

build:
```sh
cd build && cmake ..
make
```
